|            |                    |            |       |                      |           |                        |          |            | 01 正常 参考《贷款风险分类指引》（银监发〔2007〕54号），根据资产的净值、债务人的偿还能力、债务人的信用评级情况和担保情况划分正常类的资产。其中，正常类贷款是指借款人能够履行合同，没有足够理由怀疑贷款本息不能按时足额偿还的贷款。                                                                               |
